# A simple url shortener using Go, Mongodb and Traefik (load balancing)

## Tech

- Go
- MongoDB
- Traefik
- Docker

## How to run

```
make up
```
The api will be accessible in http://foo.bar.localhost

to scale up the api...

```
make scale
```
